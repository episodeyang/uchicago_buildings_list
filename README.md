UChicago Buildings List
=======================

It simply fetches from [http://registrar.uchicago.edu/page/building-abbreviations-addresses](http://registrar.uchicago.edu/page/building-abbreviations-addresses) to obtain its building list. It relies on the table on that page being consistent across the rows. It is not a smart script.
